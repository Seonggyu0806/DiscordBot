# 배틀그라운드 스쿼드 모집 봇 (PUBG Bot)

Discord 채널에서 배틀그라운드 스쿼드(4인)를 간편하게 모집하고 관리할 수 있는 디스코드 봇입니다. 

##  주요 기능
- **스쿼드 모집 생성**: 명령어를 통해 모집 게시글(임베드)과 상호작용 가능한 버튼을 생성합니다.
- **실시간 인원 현황 제공**: 버튼 클릭 시 실시간으로 참가자 목록과 현재 인원수(최대 4명)를 업데이트합니다.
- **모집 관리 기능**: 모집장(명령어 사용자)은 모집을 취소하거나 인원이 다 차기 전에 조기 출발시킬 수 있습니다.
- **자동 멘션 알림**: 4명의 스쿼드 인원이 모두 모이거나, 모집장이 조기 출발을 누르면 참가자 전원을 멘션하여 알림 메시지를 보냅니다.

##  사용 방법

### 1. 모집 시작하기
채팅창에 아래 명령어를 입력하여 모집을 시작합니다.
> **명령어:** `!배그모집`

명령어를 입력하면 봇이 모집 안내 임베드와 함께 4개의 상호작용 버튼을 출력합니다.

### 2. 버튼 기능 안내

-  **참가하기**
  - 모집에 참가합니다. 
  - 참가자 명단에 디스코드 닉네임이 등록됩니다.
  - 4명이 꽉 차면 자동으로 모집이 마감되고 참가자 전원에게 멘션 알림이 전송됩니다.

-  **내리기**
  - 참가했던 모집에서 빠집니다.
  - 단, 모집을 시작한 **모집장**은 내릴 수 없습니다. (취소를 원할 경우 '모집 취소' 사용)

-  **지금 출발**
  - 4명이 다 모이지 않아도 스쿼드를 출발시킵니다.
  - **모집장만** 클릭할 수 있으며, 최소 2명 이상이 모여야 출발이 가능합니다.
  - 실행 시 참가자 전원에게 멘션 알림이 전송됩니다.

-  **모집 취소**
  - 진행 중인 모집을 완전히 취소하고 종료합니다.
  - **모집장만** 클릭할 수 있습니다.

##  설치 및 실행 방법

1. **의존성 패키지 설치**
   ```bash
   npm install
   ```

2. **환경 변수 설정 (`.env`)**
   프로젝트 루트 디렉토리에 `.env` 파일을 생성하고 발급받은 봇 토큰을 입력합니다.
   ```env
   DISCORD_TOKEN=당신의_디스코드_봇_토큰_입력
   ```

3. **봇 실행**
   ```bash
   node index.js
   ```

*참고: 봇이 정상적으로 작동하려면 디스코드 개발자 포털(Developer Portal)에서 `MESSAGE CONTENT INTENT` 활성화가 필요합니다.*
